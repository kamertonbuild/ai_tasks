### Описание.

#### [Задача 1](src%2Fmain%2Fjava%2Fru%2Fyandex%2Fai%2Ftask1%2FHumidifierWrapper.java)
Метод **addWater** (см. [Humidifier.java](src%2Fmain%2Fjava%2Fru%2Fyandex%2Fai%2Ftask1%2FHumidifier.java)) 
получает момент времени **ti** и количество воды **vi**. Для того чтобы вычислить кол-во воды, вылившейся
к моменту **ti**, необходимо вычислить время прошедшее с момента последнего долива (**previousTi**):
и умножить на скорость выливания: (**ti** - **previousTi**) * 1. Результат вычитается из текущего значения переменной
**capacity**, которая хранит объём в увлажнителе в момент **t{i-1}**. Если значение **capacity** < 0, **capacity** 
присваивается значение **0** т.к. объем не может быть отрицательным. Далее **capacity** увеличивается на величину **vi**,
значение **previousTi** обновляется (значением переменной **ti**). Значение **capacity** содержит объём воды
в момент времени **ti** и возвращается методом **addWater**.  

**Временна сложность**: O(n) n - количество добавлений воды, т.к. время работы алгоритма зависит только кол-ва доливов.
(чтение строки и разбор строки с входными данными не учитываются)

#### [Задача 2](src%2Fmain%2Fjava%2Fru%2Fyandex%2Fai%2Ftask2%2FMaxDigital.java)
Для максимизации числа **A** нужно обновлять цифры числа **A** цифрами из числа **B** начиная со старшего разряда.
Начинать необходимо с наибольшей цифры числа **B** и продолжать по убыванию. Для этого цифры числа **B** предварительно 
сортируются. Проход по цифрам числа **А** начинается со старшего разряда. Каждая цифра из **A** (**Da**) сравнивается 
с соответствующей цифрой числа **B** (**Db**). Если **Db** > **Da**, цифра **Da** меняется на цифру **Db**. Результат записывается
в переменную **result**. Если кол-во разрядов **B** меньше кол-ва разрядов числа **A**, остаток цифр числа **A**, копируется
в конец **result**.

**Временна сложность**: O(s) s - длина строки меньшего числа, т.к. кол-во итераций цикла будет зависеть только он длинны
более короткого числа (исходим из того что копирования остатка цифр из **А** происходит за постоянное время).
